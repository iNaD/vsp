Für die Generierung der API Dokumentation wird [`RAML to HTML`](https://github.com/kevinrenskers/raml2html) genutzt.

Es kann - vorausgesetzt Nodejs ist installiert und entsprechend im Pfad eingetragen - über den Befehl `npm i -g raml2html` installiert werden.

Nun kann mit Hilfe von `raml2html "Monopoly API.raml" > API.html` die RAML-Spezifikation in ein HTML Dokument umgewandelt werden.
