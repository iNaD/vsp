/games

Manages Games

/games get post

get

Returns a list of available games.

post

Creates a new game.

get /games

Returns a list of available games.

  • Request
  • Response

Query Parameters

  • Player: (number)

    Filter for games with the given player id

    Example:

    12345
  • Round: (number)

    Filter for games with the given round

    Example:

    42

HTTP status code 200

Body

Type: application/json

Schema:

{
    "type": "object",
    "$schema": "http://json-schema.org/draft-03/schema",
    "id": "http://jsonschema.net",
    "required": true,
    "properties": {
        "games": {
            "type": "array",
            "$schema": "http://json-schema.org/draft-03/schema",
            "id": "http://jsonschema.net",
            "required": true,
            "items": { "$ref": "game" }
        }
    }
}

Example:

{
    games: [
        {
            "id": 1,
            "name": "foo",
            "round": 0,
            "players": [
                {
                    "id": 43,
                    "name": "Hans"
                }
            ]
        }
    ]
}
